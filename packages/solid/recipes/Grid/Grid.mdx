Grids of navigable items (usually involving artwork and metadata) are one of the most common UI patterns found in TV applications, often something like this grid of multicolored tiles:

<Primary of={GridStories} />

We can build this layout with a few components and styles, letting the `Row` and `Column` do the heavy lifting:

```tsx
const App = () => {
  const tileStyle = {
    width: 300,
    height: 100,
    borderRadius: 8,
    border: {
      width: 4,
      color: 0x00000000
    },
    focus: {
      border: {
        width: 4,
        color: 0xffffffff
      }
    }
  };

  return (
    <Column autofocus>
      <Row height={100}>
        <View style={tileStyle} />
        <View style={tileStyle} />
        <View style={tileStyle} />
        <View style={tileStyle} />
      </Row>
      <Row height={100}>
        <View style={tileStyle} />
        <View style={tileStyle} />
        <View style={tileStyle} />
        <View style={tileStyle} />
      </Row>
      <Row height={100}>
        <View style={tileStyle} />
        <View style={tileStyle} />
        <View style={tileStyle} />
        <View style={tileStyle} />
      </Row>
      <Row height={100}>
        <View style={tileStyle} />
        <View style={tileStyle} />
        <View style={tileStyle} />
        <View style={tileStyle} />
      </Row>
    </Column>
  );
};
```
